C. Details First

Make mobile lists useful by adding detail screens before write workflows. Lowest risk, but less business impact.

Recommendation

  • Choose A for the first phase.
  • It gives mobile a clear field-sales purpose.
  • It still reuses detail and form components internally.

Trade-off

  • A needs careful flow design for quotation and order line items.
  • B is simpler technically but feels more like a small web clone.
  • C is safest but delays create/edit value.
